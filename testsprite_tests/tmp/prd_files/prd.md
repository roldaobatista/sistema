# Sistema OS - PRD

## Overview
Sistema OS is a full-stack business management system.

## Core Modules
- Authentication (login with email/password)
- Dashboard
- Work Orders (OS)
- Customers
- Products
- Quotes
- Financial
- Commissions
- Expenses
- Service Calls
- INMETRO
- Reports
- IAM

## Tech Stack
- Frontend: React + TypeScript + Vite (port 3000)
- Backend: Laravel PHP API (port 8000)
